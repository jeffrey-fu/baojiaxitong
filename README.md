1. 写代码
2. 确认修改
    
    git diff // 显示当前改变的文件

    git status  // 当前文件状态

    git add filename  // 添加文件到本地git

    git add .  //添加所有文件操作到本地git

    git commit -m '为什么改这些文件'

    git push  //  或者git push origin master

    git branch // 显示所有分支

一般流程

git diff
git add .
git commit -m '第二次commit'
